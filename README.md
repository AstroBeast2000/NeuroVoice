NeuroVoice is a research-focused language assessment application designed to explore whether patterns in speech and verbal fluency can support the study of cognitive impairment. The application is built with Python and Streamlit and combines local speech transcription with separate machine-learning models for two established language tasks: the Cookie Theft Picture Description Task and the Semantic Verbal Fluency Test.

During the picture-description task, the participant describes the Cookie Theft image in their own words. NeuroVoice records the response, transcribes the audio locally using the Whisper small speech-recognition model, and analyzes the transcript using an ensemble of word-based, character-based, and linguistic-feature classifiers. The model examines patterns such as vocabulary use, sentence construction, information content, and broader language structure.

During the Semantic Verbal Fluency Test, the participant names as many animals as possible for at least sixty seconds. The application analyzes the resulting transcript using a task-specific model and calculates understandable response characteristics, including the total number of recognized items, the number of unique animal names, repetitions, and the unique-name ratio.

NeuroVoice includes recording-duration requirements, audio preprocessing, transcription-quality checks, and safeguards against common Whisper hallucinations such as artificial phrase repetition or unrelated boilerplate text. Unreliable transcripts are rejected before they are submitted to the classification models. Results are displayed using plain-language explanations alongside the model score, decision threshold, flagged status, transcript, component-model outputs, and relevant limitations.

Users can also download a PDF report containing both task results, transcripts, model scores, fluency characteristics, interpretation guidance, and a research-use disclaimer. The application does not average the two model scores because each model was trained for a different language task.

NeuroVoice is intended for research, education, and technical demonstration only. Its outputs are experimental and must not be interpreted as a medical diagnosis, clinical screening result, disease probability, or measurement of impairment severity. The current models require additional external validation across larger and more diverse populations before any potential clinical use. Recording conditions, microphone quality, speech-recognition accuracy, dataset differences, and model-training limitations may all influence the generated results.
